Chapter 1 Introduction

These notes are based on “Regression and other stories” by Gelman, Hill, and Vehtari (2020). Given the different background in statistics that we potentially have, this notes aim to have a standardized tool for tutorials.

Finally, these notes aim to cover the same content as weekly lectures.
